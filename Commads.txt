pwd - показывает путь к папке, в которой я нахожусь 
ls - просмотр содержимого каталого ( -R список файлов в подкаталогах -a скрытые файлы )
cat - вывод содержимого файла в командной строке
cp - копирует файлы из одного каталога в другой 
mv - перемещение и переименование файлов
mkdir - создание новой папки
rmid - удаление папки
rm - удаление файла ( -r удаление каталога со всем содрежимым )
find - поиск файла папке
df - получение отчета об использовании дискового пространства в системе
man - просмотр документации команды
